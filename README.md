# SpeakMyTexts
SpeakMyTexts turns quiet text into living voices. made with a vision to help students, especially those who cant see, read, learn and explore more.

This project converts books, PDFs, notes, typed texts into clear speech using gTTS.

# Features
1. Convert text to speech
2. Read normal pdf
3. Multilingual support
4. simple UI

# Tech Used
Python, Tkinter(for GUI), gTTS(Google text to speech), pypdf(To read pdfs).
